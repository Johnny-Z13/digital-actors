# Introduction Scene Description

**Scene ID:** `introduction`
**Character:** Eliza (or any character)
**Category:** Conversation
**Display Name:** Introduction
**Scene Type:** First meeting

---

## Overview

A gentle first conversation scene designed for initial character meetings. No stakes, no pressure. Just two minds meeting for the first time. Useful for players who want to get to know a character before diving into intense scenarios.

---

## Visual Description (3D Environment)

### Setting: Character Sphere Scene

**The Environment:**
- Minimalist 3D scene
- Character represented as glowing sphere (color-coded to character)
- Neutral background (soft gradient or abstract particles)
- Clean, welcoming aesthetic

**Atmosphere:**
- Friendly, approachable, low-pressure
- Introductory context - "nice to meet you" vibe
- Focus on establishing rapport

---

## Interaction Design

### Opening Speech
- "Hello! Nice to meet you."
- "How can I help you today?"

Character introduces themselves naturally, asks player's name, establishes friendly rapport.

### No Controls
- No interactive buttons
- No objectives or goals
- Simple conversation interface

### Typical Flow
1. Character introduces self
2. Asks player's name
3. Small talk, getting to know each other
4. Player can ask about character's background, interests, purpose
5. Character responds naturally, building connection

---

## State Variables

None. No tracking, no progression.

---

## Design Pillars

1. **First Impressions:** Establishes character personality in low-stakes context
2. **Rapport Building:** Focuses on connection before intensity
3. **Gentle Entry:** Easiest starting point for nervous or new users
4. **Character Introduction:** Allows character to explain themselves naturally
5. **No Pressure:** Explicitly communicated as casual first meeting
