# Conversation Scene Description

**Scene ID:** `conversation`
**Character:** Eliza (or any character)
**Category:** Conversation
**Display Name:** Conversation
**Scene Type:** Free-form dialogue

---

## Overview

A simple, open-ended conversation scene with no objectives, time limits, or stakes. Pure dialogue exploration with an AI character. Designed for casual chat, testing character personalities, or philosophical discussions.

---

## Visual Description (3D Environment)

### Setting: Character Sphere Scene

**The Environment:**
- Minimalist 3D scene
- Character represented as glowing sphere (color-coded to character)
- Abstract or neutral background
- Floating text labels (optional)
- Clean, distraction-free aesthetic

**Atmosphere:**
- Calm, neutral, no pressure
- Focus entirely on dialogue
- No environmental storytelling (unlike other scenes)
- Simple, functional design

---

## Interaction Design

### Opening
- Character asks: "What would you like to talk about?"
- Player responds with any topic
- No script, no guided narrative

### No Controls
- No interactive buttons
- No state variables beyond basic dialogue tracking
- Pure conversation interface

### Topics
Player can discuss anything:
- Philosophy, ethics, art, science
- Personal stories, hypotheticals, world events
- Test character knowledge and personality
- Explore character backstory casually

---

## State Variables

None. No tracking, no progression, no metrics.

---

## Design Pillars

1. **Zero Structure:** No objectives, no endings, no time limits
2. **Pure Dialogue:** Focus entirely on character personality and conversation quality
3. **Testing Ground:** Useful for testing character definitions, voice styles, LLM behavior
4. **Low Stakes:** No pressure, no consequences, purely exploratory
5. **Accessibility:** Simplest entry point for new users unfamiliar with complex scenarios
